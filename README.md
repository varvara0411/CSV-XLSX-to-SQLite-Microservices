# CSV-XLSX-to-SQLite-Microservices

## **Обзор проекта**

Данный репозиторий содержит две реализации микросервисов для загрузки CSV и Excel файлов с автоматическим созданием таблиц в SQLite базе данных. Оба сервиса разработаны в соответствии с требованием: **без использования библиотек, предоставляющих готовые методы для создания таблицы БД из файла**.

Сервисы автоматически определяют типы колонок (INTEGER, REAL, TEXT, TIMESTAMP) путем анализа фактических данных в каждой колонке, генерируют соответствующие CREATE TABLE запросы и вставляют данные построчно с использованием параметризованных запросов для безопасности.

В репозитории представлены две реализации:
- **Flask версия** - синхронная реализация на фреймворке Flask
- **FastAPI версия** - асинхронная реализация на фреймворке FastAPI с автоматической Swagger документацией

## **Flask реализация**

Flask версия находится в директории `/flask`. Она использует отдельный класс `DatabaseManager` для инкапсуляции всех операций с базой данных.

### **Ключевые особенности:**
- **HTML документация** на корневом эндпоинте (`/`)
- **Загрузка файлов** с автоматическим определением кодировки и разделителя для CSV
- **Поддержка Excel** через `pandas.read_excel`
- **Ручное определение типов** на основе анализа данных
- **Последовательное именование таблиц** (`table_1`, `table_2`, и т.д.)
- **Эндпоинт для списка всех таблиц** (`/tables`)
- **Эндпоинт для просмотра содержимого таблицы** с красивым JSON форматированием (`/table/{table_name}`)
- **Проверка работоспособности** (health check через эндпоинт `/health`)
- **Ограничение размера файла 16MB** для безопасности

## **FastAPI реализация**

FastAPI версия находится в директории `/fastapi`. Она использует асинхронные возможности FastAPI и предоставляет автоматическую Swagger документацию.

### **Ключевые особенности:**
- **Автоматическая Swagger UI** по адресу `/docs`
- **Красивое JSON форматирование** всех ответов с отступами и поддержкой Unicode
- **Парсинг CSV** с использованием встроенного модуля `csv`
- **Поддержка Excel** через `pandas.read_excel`
- **Ручное определение типов** на основе анализа данных
- **Последовательное именование таблиц** (`table_1`, `table_2`, и т.д.)
- **Эндпоинт для списка таблиц** (`/tables`)
- **Эндпоинт для просмотра таблицы** (`/table/{table_name}`)
- **Глобальный обработчик ошибок** для единообразных ответов
- **Асинхронное чтение файлов** для лучшей производительности

## **Общие возможности (обе реализации)**

Оба сервиса имеют следующие общие характеристики:

- **SQLite база данных** (`database.db` в директории instance для Flask, в текущей директории для FastAPI)
- **Автоматическое определение типов колонок** (INTEGER, REAL, TEXT, TIMESTAMP)
- **Поддержка CSV и XLSX файлов**
- **Определение кодировки CSV** (UTF-8, CP1251, Windows-1251, Latin-1)
- **Определение разделителя CSV** (запятая, точка с запятой, табуляция)
- **Безопасные имена таблиц** (пробелы и дефисы заменяются на подчеркивания в Flask версии, имена заключаются в двойные кавычки в FastAPI версии)
- **Параметризованные запросы** для защиты от SQL-инъекций
- **Транзакции** с commit/rollback
- **Проверка на пустые файлы**
- **Проверка на отсутствие данных** (файлы только с заголовками отклоняются)
- **RESTful дизайн API**

## **Логика работы**

### **Процесс загрузки файла:**
1. Клиент отправляет файл через POST запрос на `/upload`
2. Сервис проверяет расширение файла и его содержимое
3. Для CSV файлов перебираются различные кодировки и разделители
4. Для Excel файлов pandas читает данные из памяти
5. Типы колонок определяются путем анализа всех значений
6. Генерируется новое имя таблицы (`table_X`, где X - следующий доступный номер)
7. Выполняется CREATE TABLE запрос с соответствующими типами колонок
8. Все строки вставляются с использованием параметризованных запросов
9. В ответ возвращается информация о созданной таблице

### **Логика определения типов:**
- Если хотя бы одно значение в колонке TEXT → вся колонка становится TEXT
- Иначе если есть значение TIMESTAMP → вся колонка становится TIMESTAMP
- Иначе если есть значение REAL → вся колонка становится REAL
- Иначе → INTEGER

## **Установка и запуск**

### **Требования**
- Python 3.8 или выше
- pip (менеджер пакетов Python)

### **Flask версия**

```bash
# Перейти в директорию flask
cd flask

# Создать виртуальное окружение
python3 -m venv venv

# Активировать виртуальное окружение
# На Windows:
venv\Scripts\activate
# На macOS/Linux:
source venv/bin/activate

# Установить зависимости
pip install -r requirements.txt

# Запустить приложение
python3 app.py
```

Flask сервер запустится по адресу `http://localhost:5000`

### **FastAPI версия**

```bash
# Перейти в директорию fastapi
cd fastapi

# Создать виртуальное окружение
python3 -m venv venv

# Активировать виртуальное окружение
# На Windows:
venv\Scripts\activate
# На macOS/Linux:
source venv/bin/activate

# Установить зависимости
pip install -r requirements.txt

# Запустить приложение с uvicorn
uvicorn main:app --reload
```

FastAPI сервер запустится по адресу `http://localhost:8000`

## **Тестирование сервисов**

### **Через браузер**

**Flask:**
- Откройте `http://localhost:5000` - документация API
- `http://localhost:5000/health` - проверка работоспособности
- `http://localhost:5000/tables` - список таблиц

**FastAPI:**
- Откройте `http://localhost:8000` - приветственное сообщение
- `http://localhost:8000/docs` - Swagger документация (для тестирования)
- `http://localhost:8000/health` - проверка работоспособности
- `http://localhost:8000/tables` - список таблиц

### **Через curl**

**Загрузка CSV файла:**
```bash
curl -X POST http://localhost:5000/upload -F "file=@data.csv"
# или для FastAPI (порт 8000)
curl -X POST http://localhost:8000/upload -F "file=@data.csv"
```

**Загрузка Excel файла:**
```bash
curl -X POST http://localhost:5000/upload -F "file=@data.xlsx"
# или для FastAPI (порт 8000)
curl -X POST http://localhost:8000/upload -F "file=@data.xlsx"
```

**Получение списка таблиц:**
```bash
curl http://localhost:5000/tables
# или для FastAPI (порт 8000)
curl http://localhost:8000/tables
```

**Просмотр содержимого таблицы:**
```bash
curl http://localhost:5000/table/table_1
# или для FastAPI (порт 8000)
curl http://localhost:8000/table/table_1
```

**Проверка работоспособности:**
```bash
curl http://localhost:5000/health
```

## **Важные замечания**

1. **Безопасность**: Оба сервиса используют параметризованные запросы для предотвращения SQL-инъекций
2. **Транзакции**: В случае ошибки во время вставки данных все изменения откатываются
3. **Кодировки**: Поддерживаются UTF-8, CP1251, Windows-1251, Latin-1 для CSV файлов
4. **Разделители**: Автоматически определяются запятая, точка с запятой и табуляция
